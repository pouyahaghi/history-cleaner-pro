# History Cleaner Pro

History Cleaner Pro is a Chrome extension that provides advanced history cleaning capabilities, including cache deletion, scheduling, and one-click cleaning.

## Features

*   **Quick Clear:** Clears the browsing history of the last hour.
*   **Full Clean:** Clears all browsing history, cache, and cookies.
*   **Targeted Cleaning:** Clears history based on specific URLs or keywords.
*   **Scheduled Cleaning:** Automatically cleans history at regular intervals.
*   **Startup Cleaning:** Cleans history when Chrome starts.
*   **Advanced Cleaning:** Clears specific types of browsing data, such as cache, cookies, and local storage.

## Usage Examples

### Quick Clear

To quickly clear the browsing history of the last hour, click the "Quick Clear" button in the popup.

### Full Clean

To clear all browsing history, cache, and cookies, click the "Full Clean" button in the popup. A confirmation dialog will appear before the action is performed.

### Targeted Cleaning

#### Clear by URL

1.  Enter the URL in the "Clear by URL" input field.
2.  Click the "Clear History for URL" button.

#### Clear by Keyword

1.  Enter the keyword in the "Clear by Keyword" input field.
2.  Click the "Clear History for Keyword" button.

#### Clear Multiple URLs

1.  Enter the URLs in the "Clear Multiple URLs" textarea, one URL per line.
2.  Click the "Clear All Listed URLs" button.

### Scheduled Cleaning

1.  Go to the settings page by clicking the settings button in the popup.
2.  Enable scheduled cleaning by checking the "Enable automatic cleaning" checkbox.
3.  Select the desired cleaning interval from the "Interval" dropdown.
4.  Choose the data types to clean by checking the corresponding checkboxes.
5.  Click the "Save Schedule" button.

### Startup Cleaning

1.  Go to the settings page by clicking the settings button in the popup.
2.  Enable startup cleaning by checking the "Clean on browser startup" checkbox.
3.  Choose the data types to clean by checking the corresponding checkboxes.
4.  Click the "Save Startup Settings" button.

### Advanced Cleaning

1.  Select the data types to clear by checking the corresponding checkboxes in the "Clear Cache & Cookies" section.
2.  Click the "Clear Selected Data" button.

## Configuration Details

### Settings Page

The settings page can be accessed by clicking the settings button in the popup. The settings page allows you to configure the following options:

*   **Scheduled Cleaning:**
    *   Enable/disable scheduled cleaning.
    *   Cleaning interval.
    *   Data types to clean (browsing history, cache, cookies).
*   **Startup Cleaning:**
    *   Enable/disable startup cleaning.
    *   Data types to clean (browsing history, cache, cookies).
*   **Quick Actions Settings**
    *   Quick Clear Action: What Quick Clear button should do
    *   Full Clean Action: What Full Clean button should do
*   **Confirmation Settings**
    *   Show confirmation for Quick Clear
    *   Show confirmation for Full Clean
    *   Show confirmation for Clear All History

## Dependencies

This extension does not have any external dependencies.
